Things You Can Do with Google Classroom

For a teacher

  1. Invite students to join the classroom
  2. Prepare teaching materials for a lesson
  3. Teach online with Google Meet
  4. Create homework assignments
  5. Assign homework to all or selected students of the class
  6. Check and grade the submitted homework
  7. Answer questions and communicate with any student privately
  8. Make announcements to the class online
  9. ... ...

For a student

  1. Do homework online
  2. Ask questions and communicate with the teacher privately
  3. Submit/re-submit the assignment
  4. ... ...

Create a Classroom (Admin/Teacher)

  1. From the admin console, set the classroom membership to allow personal Gmail account owners to join the classroom. (https://support.google.com/edu/classroom/answer/6173514) for more information.
  2. Login as a teacher (c9@greatwallchineseacademy.org)
  3. Go to Google Classroom (https://classroom.google.com/h)
  4. Create a new classroom (click on the plus sign)

When the process is completed, the teacher is the primary teacher (owner) of the classroom and you may invite other teachers to the classroom as co-teachers. And the classroom now is ready to use.

Invite Students to Your Classroom (Teacher)

First of all, all the students must have a Google account (Gmail) to join the classroom. If anyone doesn’t have a Google account, ask him/her to create one first.

There are three ways to invite students:

  1. By invitation. From the classroom home page, click on People and then the “+” icon
  2. By class code. From the classroom homepage, copy the class code and give it to the students. They need to login into their Google account, go to https://classroom.google.com/, and click the “+” icon to join the classroom by the class code.
  3. By invite link. From the classroom homepage, click on the Display icon, copy the invite link, and give the link to the students. They can simply join the classroom from a browser using the link.

Create and Assign an Assignment (Teacher)

An assignment is an instructional document set for the students to do one piece of homework. It may contain a list of Google Docs documents, YouTube video clips, audio clips, and some other Google files like slides, spreadsheets, etc. When an assignment is assigned, the students will be able to follow the instructions to do the assignment.

Here are the steps to create a new assignment:

  1. Go to Classwork and click on the Create icon and select “Assignment”
  2. Create a Google Docs file and other types of Google files for detailed instructions.
  3. Add audio clips, YouTube video clips, and other types of files as attachments.

Note that for most of the assignments, they only need one Google Docs file as an instruction. All others are optional.

If you need to make a copy of the assignment for each student, e.g. everyone needs to create a new PPT for a presentation, you may choose a file attribute to do so. If the “make a copy for each student” is selected, a copy will be made for each student when the assignment gets assigned. The students can work on their copies from the assignment page.

Review the instructions and attachments. When they are ready, you may assign the assignment to all the students or selected ones. The assignment can be done manually or by schedule. When it gets assigned, the students will be notified by email and they can start doing homework immediately. After the assignment, you may use this page to un-assign or reassign this assignment to the selected students. (check or uncheck students)

Check Homework Online (Teacher)

When an assignment is turned in, you can check the assignment by clicking on it.

You could make comments, suggestions, and anything you want to tell your student about the homework assignment he/she just turned in. And return it to the student with a grade if everything is okay. Or return it without a grade if you want the student to make some changes. The student will get notified when the assignment is returned to him/her. If he/she needs to make some changes as instructed, he/she will re-submit the assignment.

Do Homework Online (Student)

Receives an email from the teacher and click the Open button to get started, or go to the Classroom homepage and select an assignment to get started.

Read the instructions and then start working on the assignment. You may

  1. Click on the assignment to start working on it.
  2. Click on the attachment to listen to the audio or watch the video clips.
  3. Click on the “Add or create” link to add an attachment to your assignment.
  4. Add a private comment to communicate with your teacher.
  5. And finally, click on the “Turn in” button to turn in the assignment.

If everything is good, your teacher will return your assignment WITH a grade. If your teacher wants you to make some changes, he/she will return the assignment WITHOUT a grade. The teacher might have some comments in the assignment to instruct you on what kind of changes you need to make. In that case, you need to make the necessary changes and resubmit the assignment. While making changes, you need to accept or reject your teacher's suggestions.

Note that when an assignment is turned in, the teacher will not get notified until he/she checks the status of your assignment. You may make a simple private comment while turning in the assignment so that the teacher will be notified immediately.
