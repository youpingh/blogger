亡羊补牢

2010-4-7

五彩的手一哆嗦，五篇精彩博文瞬时踪影全无。急得五彩嘴上直起泡，抱起双手向四方作揖：大小姑奶奶，老少爷们，行行好，伸把手，帮俺一把吧。自己的博文有底儿，可网友的评论我上哪找回来呀！隔着网都听出哭腔来了。

谢天谢地，得高人帮忙，五彩把大部分丢了的文章及评论都找回来了。算是不幸之中的万幸。用五彩的话讲“能找到这些，俺已经很知足了。”五彩还说：“蛮夷，昨天文章丢了后，俺就想起你的变天账，后悔怎么没弄一份。”

其实我那份“变天账”也救不了这样的急，因为那里记得都是我在别人那里留的言，而不是别人在我这留的言。为了保证不丢失任何人在我这留下的至理名言，今天下午我做了一次蛮夷后备，把大家对我的忠告，讽刺，批评和表扬都留了个底儿。不然的话，万一哪天我的手跟五彩的手似的也哆嗦了一下，真着不起那急呀。

做起来很简单。走到你想留底儿的那篇文章，用五彩的例子：忆苦思甜：大饼子，窝窝头和花卷，然后用 File -> Save page as … 保留那篇文章到你的硬盘上。（文章和所有的评论都会存起来的）。注意，用 Save page as 时，保留文件的类型要选用 “Web Page，complete”。重复这个步骤，直到所有的文章都做完。

简单易行，有备无患。
