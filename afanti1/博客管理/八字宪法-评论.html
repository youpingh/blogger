简单的文字评论

包括我自己在内的很多博主都是从发表些简单的文字评论开始的。发表评论的那个地方和发表日志里的那个“粘贴为无格式文本”的小窗口差不多，不管你往上贴什么，所有附带的格式控制字符都会被去掉。所以不用担心那些控制字符会捣乱，尽管放心贴就好了。

声情并茂的评论

如果想在评论中变换字体，颜色，插图，加录像，放音乐，怎么办？那个看似简单的留言窗口实际上是一个功能齐全的 HTML 编辑窗口（我相信所有的 HTML 编辑工作是在万维后台服务器上做的，但这不妨碍我们把这个小窗口看成一个 HTML 编辑器），所以所有（或大多数）贴在那个窗口上的 HTML 编码，都会起到控制显示的作用。

方法一，我称之为暗渡陈仓法，是从转悠那抄来的。 转悠在她的文章中解释非常清楚，我就不在这重复了。请看她的文章： 如何在跟贴中插入图片？

修改删除评论

这大概是一个相当普遍的技术问题，我自己就碰到过好几次。如果在别人的博客中写评论时写错了字，那你就认倒霉吧。除非你请博主帮忙删掉那条评论，否则白字先生的名字你就得背一辈子了。所以拍板砖的时候一定要特别小心，千万别在那会儿露怯。如果要在你自己的博客中写错了字，你可以把那条评论删了，然后重新发一条改过的。我注意到万维的删除评论的功能不太好使，“删除这条评论”的链接时有时无，常有神龙见首不见尾的感觉。前些时候我花了小一个钟头，终于找到了一个可行的办法在自己的博客中删除评论。具体步骤如下：

A．从你自己博客的首页登录，
B．走到你要删除评论的文章，比如说：《我是流氓我怕谁》
C．在文章后随便加一条评论，比如说：我不是流氓我也不怕谁。

发表这条评论后，那个不听话的“删除这条评论”的链接就会乖乖的出来了，用它就可以删掉任何评论了。注意，有时被删掉的评论不会马上就从文章的页面中消失。估计是因为这些评论还在万维服务器的暂存（Cache）里。耐心等一会儿（十到二十分钟左右）再看看以确定评论确实已被删除。
